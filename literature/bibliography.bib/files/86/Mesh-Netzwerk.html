Definition

Mesh-Netzwerk

Veröffentlicht von: Margaret Rouse
Whatis.com/de
  • Teilen Sie diesen Inhalt mit Ihrem Netzwerk:

Eine Mesh-Netzwerk-Topologie (Maschen-Netzwerk) ist ein dezentralisiertes Netzwerkdesign, bei dem jeder Node oder Knotenpunkt in einem Netzwerk mit mindestens zwei anderen verbunden ist. Von Mesh-Netzwerken erwartet man, dass sie zukünftig vor allem beim Internet der Dinge (IoT / Internet of Things) eine wichtige Rolle spielen.

Bei einer Stern-Topologie benötigen die Nodes Router, der Internet-Services zur Verfügung stellt. Beim Mesh-Netzwerk können die Nodes dagegen direkt miteinander kommunizieren und brauchen dafür keine aktive Internetverbindung. Ein großer Vorteil dieser dezentralisierten Topologie ist, dass es keinen so genannten Single Point of Failure (SpoF) gibt. Sollte ein Node aus irgendwelchen Gründen nicht mehr erreichbar sein, können die anderen weiterhin miteinander kommunizieren. Das ist entweder direkt oder durch weitere Nodes möglich, die als Mittelsmänner fungieren.

Mesh-Netzwerke können eine vollständige Mesh-Topologie oder nur eine Teil-Mesh-Topologie im Einsatz haben. In einer vollständigen Mesh-Topologie ist jeder Netzwerk-Node mit allen anderen Nodes im Netzwerk verbunden. Bei einer Teil-Mesh-Topologie verbindet sich mindestens ein Node direkt mit jedem anderen Node und andere verbinden sich möglicherweise nur mit diesem Node. Sie alle tauschen Daten in regelmäßigen Abständen aus.

In der Vergangenheit waren Mesh-Netzwerke immer drahtgebunden. Aus diesem Grund konnte die Topologie teuer und kompliziert sein, da man jeden Node physisch mit jedem anderen Knoten verbinden musste. Heutzutage gibt es signifikante Fortschritte bei der drahtlosen Kommunikation und bei Kurzstrecken-WPAN (Wireless Personal Network). Diese Spezifikationen haben die physischen und finanziellen Barrieren ausgehebelt.

Die nachfolgende Illustration zeigt ein vollständiges Mesh-Netzwerk mit sechs Nodes. Jeder Node wird als Kugel dargestellt und die Verbindungen als gerade Linien.

 

Die zweite Illustration stellt ein Teil-Mesh-Netzwerk mit sechs Nodes dar. Jeder Node wird als Kugel dargestellt und die Verbindungen wiederum als gerade Linien. Die Verbindungen in einem vollständigen oder Teil-Mesh-Netzwerk können entweder drahtgebunden oder drahtlos sein.
